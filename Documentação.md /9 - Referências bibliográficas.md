# Referências Bibliográficas

Este projeto foi desenvolvido com base nos códigos Python fornecidos, que agora utilizam uma arquitetura modular e uma interface gráfica de usuário (GUI). As seguintes bibliotecas e conceitos foram empregados:

•
Python Standard Library: Para funcionalidades básicas e módulos como random e os.

•
Python Software Foundation. (s.d.). The Python Standard Library. Disponível em: https://docs.python.org/3/library/



•
CustomTkinter: Utilizado para a criação da interface gráfica de usuário (GUI) com um visual moderno.

•
CustomTkinter. (s.d.). Documentation. Disponível em: https://customtkinter.tomschimansky.com/documentation/



•
Pillow (PIL Fork): Utilizado para manipulação de imagens, especificamente para carregar a imagem dark_crown.png na GUI.

•
Pillow. (s.d.). Documentation. Disponível em: https://pillow.readthedocs.io/en/stable/



•
Conceito Model-View-Controller (MVC): A arquitetura do projeto segue o padrão MVC para separar as responsabilidades de dados (Model), lógica de negócio (Controller) e interface de usuário (View).

•
Gamma, E., Helm, R., Johnson, R., & Vlissides, J. (1994). Design Patterns: Elements of Reusable Object-Oriented Software. Addison-Wesley. (Referência conceitual para MVC)
